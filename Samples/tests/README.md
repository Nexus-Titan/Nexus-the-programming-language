# Nexus V4 Test Suite

Scripts for verifying the functionality and performance of the Nexus V4 engine.

## Key Tests

- master_check.nx: Comprehensive test of all 20 language libraries.
- lib_check.nx: Health check for core production modules.
- dashboard.nx: Test for the system monitoring and GUI subsystem.
- security_audit.nx: Integrity check for local script repositories.
