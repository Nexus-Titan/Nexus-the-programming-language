# Nexus V4 Standard Libraries

This directory contains pre-built modules for the Nexus language.

## Modules

- fs.nx: Advanced filesystem management.
- logger.nx: Standardized application logging.
- http_client.nx: REST API interaction utilities.
- json_db.nx: Local JSON-based data persistence.
- py_suite.py: Python-based data science and utility extensions.
- nx_suite.nx: Native Nexus graphics and web crawling tools.

## Usage

Import a module using the following syntax:
import "module_name.nx"
