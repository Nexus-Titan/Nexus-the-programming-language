# Nexus V4 Unified Engine

This directory contains the core interpreters and libraries for the Nexus V4 standard.

## Components

- interpreter.py: Reference Python implementation with full library support.
- interpreter.cpp: High-performance C++ implementation.
- interpreter.js: Native JavaScript implementation for web environments.

## Directory Layout

- libs/: Standard and extension libraries.
- tests/: Suite of check scripts to verify engine integrity.
- libs/src/: Source code for compiled native libraries (C, CPP, Rust, ASM, Java).
